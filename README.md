**Title:** Free customizable mug design files for personalized gifts - Sublimation ready (CDR + SVG)

I wanted to share something I created a few years ago that might help those of you struggling with gift ideas. I designed some minimalist mugs that feature a cartoon-style mouth or smile on the left side and a person's name on the right side in a modern handwritten font. The idea is that when the person holds the mug with their right hand and brings it to their mouth, the cartoon smile aligns with their face, creating a fun effect while the personalized name is visible from the front.

I'm sharing these design files completely free for anyone who wants to use them. You'll find:
- CDR file (CorelDRAW)
- SVG file (compatible with most design software)
- Mockup images showing how it looks when printed

**How to use:**
1. Download the files
2. Customize the name in the design (if needed)
3. Take the files to any local print shop that offers sublimation printing
4. Print on a standard white ceramic mug

This is perfect for birthdays, holidays, or just thoughtful personalized gifts. The designs are minimalist and work well on plain white mugs.

Feel free to ask if you have any questions about the files or the printing process!
